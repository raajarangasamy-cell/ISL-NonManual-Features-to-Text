Dataset Description

The dataset for this project consists of 
images and videos representing the 
non-manual features of Indian Sign 
Language (ISL). These features include 
facial expressions, eye movements, and 
head movements that convey important 
information during sign language 
communication. The data can be collected
from publicly available ISL resources,
educational videos, and sample recordings.
The dataset is stored in image formats 
such as JPG and PNG, and video formats 
such as MP4. This data is used for 
feature extraction, analysis, and 
conversion of non-manual sign language 
features into meaningful text.
