# Methodology

## Step 1: Data Collection
Capture facial expressions and head movements using a webcam.

## Step 2: Feature Detection
Identify non-manual features such as eyebrow movements, eye movements, and head gestures.

## Step 3: Data Processing
Analyze the captured features and extract meaningful patterns.

## Step 4: Text Conversion
Map the detected patterns to corresponding text outputs.

## Step 5: Output Display
Display the generated text to the user.
