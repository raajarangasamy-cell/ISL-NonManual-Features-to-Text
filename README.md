Capturing Non-Manual Features of Indian Sign Language and Converting it into Text
Project Overview
This project aims to capture non-manual features of Indian Sign Language (ISL), such as facial expressions and head movements, and convert them into meaningful text.
Objectives
Capture facial expressions using a webcam.
Detect non-manual features of ISL.
Convert detected expressions into text.
Display the generated text to the user.
Technologies Used
Python
OpenCV
MediaPipe
GitHub
