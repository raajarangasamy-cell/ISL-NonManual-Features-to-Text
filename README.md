Capturing Non-Manual Features of Indian Sign Language and Converting Them into Text
Project Overview
This project aims to capture non-manual features of Indian Sign Language (ISL), such as facial expressions, head movements, and eye movements, and convert them into meaningful text.
Objectives
Detect non-manual ISL features.
Extract facial and head movement information.
Convert detected features into text.
Improve communication accessibility.
Technology Stack
Python
OpenCV
MediaPipe
NumPy
Pandas
GitHub
